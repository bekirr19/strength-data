# 🏋️ Gym Tracker - React App

Modern, kullanıcı dostu antrenman takip uygulaması.

## ✨ Özellikler

- 📅 **Haftalık Takvim Görünümü**: Sağa-sola kaydırarak geçiş yapın
- 📆 **Aylık Takvim Modal**: Antrenman yapılan günler işaretli
- 💪 **Egzersiz Yönetimi**: Ekleme, arama, sıralama
- 📊 **İlerleme Takibi**: PR ve tonaj metrikleri
- 🌙 **Dark Mode**: Göz dostu koyu tema
- 💾 **LocalStorage**: Veriler tarayıcıda saklanır

## 🚀 Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Production build
npm run build
```

## 📱 Kullanım

1. **Ana Sayfa**: Günlük antrenmanlarınızı görün
2. **Takvim**: Üst menüden takvim ikonuna tıklayın
3. **Hafta Şeridi**: Sağa-sola kaydırarak haftalar arası geçiş yapın
4. **Egzersiz Detay**: Herhangi bir egzersize tıklayarak ilerleme görebilirsiniz

## 🛠️ Teknolojiler

- React 18
- React Router v6
- Tailwind CSS
- Vite
- LocalStorage API
