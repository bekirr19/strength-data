Labelled metric tile — Best weight, Strength change, session totals.

```jsx
<StatCard tone="gold" icon={<Trophy size={16}/>} label="Best" value="100 kg" sub="12 Nov 2025" />
<StatCard icon={<TrendingUp size={16}/>} label="Strength change (1RM)" value="+4%" sub="vs last 3 sessions" trend="up" />
```

`tone="gold"` for the personal-best highlight; `trend="up"|"down"` colours the value green/red.
