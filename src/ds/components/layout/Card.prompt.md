White rounded surface (rounded-3xl) with a soft shadow — the base container for everything.

```jsx
<Card pad="lg">…</Card>
<Card interactive onClick={open}>Tappable row</Card>
<Card tint="accent" pad="md">Workout fuel</Card>
```

`pad`: none/sm/md/lg/xl. `interactive` adds a hover lift. `tint`: `accent`, `sunken`, or `gold` for subtle panels (fuel, notes, highlights).
